Fixed OCR Demo - Proper Error Handling

This demo shows how to properly handle OCR responses to avoid the "Cannot read properties of undefined (reading 'split')" error.

Code Example - Before (Broken):

// This causes the error:
const lines = data.extracted_text.split('\n'); // Error if extracted_text is undefined

Code Example - After (Fixed):

// Proper error handling:
if (data && data.success && data.extracted_text) {
    const lines = data.extracted_text.split('\n');
    // Process lines safely...
} else {
    // Handle error case
    console.error('OCR failed or no text extracted');
    showError(data?.error || 'Unknown error occurred');
}
